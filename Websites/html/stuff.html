Erläuterungen

Tag
In HTML werden Webinhalte durch sogenannte Tags strukturiert. Ein Tag besteht aus spitzen Klammern und einem Namen, z.B. <head> oder <p>. Tags können verschachtelt werden, wodurch sie eine Hierarchie bilden.
Elemente
Ein Element besteht aus einem öffnenden und einem schließenden Tag sowie dem Inhalt dazwischen. Zum Beispiel besteht das <p>-Element aus einem <p>-Öffnungstag, einem Textinhalt und einem <p>-Schließtag. Es gibt auch sogenannte leere Elemente, die keinen Inhalt haben und nur aus einem einzigen öffnenden Tag bestehen, z.B. <img/>.
Attribute
Attribute sind zusätzliche Informationen, die man Tags hinzufügen kann, um ihr Verhalten oder Aussehen zu ändern. Attribute werden innerhalb des öffnenden Tags angegeben und bestehen aus einem Attributnamen und einem Attributwert, z.B. <img src="bild.jpg" alt="Ein schönes Bild">. Das src-Attribut gibt hier den Pfad zur Bilddatei an und das alt-Attribut beschreibt das Bild für den Fall, dass es nicht angezeigt werden kann oder der Benutzer eine Screenreader-Software verwendet.
ID/Class
ID und Class sind Attribute, die verwendet werden, um ein HTML-Element zu identifizieren oder ihm eine Klasse zuzuweisen. Das ID-Attribut definiert einen eindeutigen Bezeichner für ein Element, während das Class-Attribut verwendet wird, um Elemente zu gruppieren und ihnen gemeinsame Eigenschaften zuzuweisen.
div Elemente
Das div-Element ist ein Container-Element, das verwendet wird, um andere HTML-Elemente zu gruppieren und zu strukturieren. Es hat keine spezifische Bedeutung und kann für jede Art von Inhalt verwendet werden. Das div-Element kann mit CSS gestaltet werden, um das Aussehen und Verhalten von Elementen auf der Seite zu steuern.
Box Model
Das CSS Box-Modell beschreibt, wie ein HTML-Element in eine rechteckige Box umgewandelt wird und wie der Inhalt, die Polsterung (Padding), die Rahmen (Border) und die Marge (Margin) dieser Box dargestellt werden. Jedes HTML-Element wird als Box behandelt, wobei die Box im Wesentlichen aus vier Schichten besteht:
  • Content (Inhalt) - Der eigentliche Inhalt des Elements, wie Text, Bilder usw.
  • Padding (Polsterung) - Ein Bereich um den Inhalt, der Abstand zwischen dem Inhalt und dem Rand des Elements schafft.
  • Border (Rahmen) - Eine Linie um den Inhalt und die Polsterung.
  • Margin (Marge) - Der Abstand zwischen dem Element und seinen Nachbarn.
    • Das Box-Modell wird verwendet, um die Größe und Position von HTML-Elementen im Layout der Website zu kontrollieren. Die Größe einer Box wird durch die Summe aus dem Inhalt, der Polsterung, dem Rahmen und der Marge bestimmt. Wenn die Größe eines Elements geändert wird, ändert sich automatisch auch die Größe aller vier Schichten der Box. Mithilfe von CSS-Regeln können diese Schichten individuell gestaltet und angepasst werden, um das Aussehen und Verhalten von HTML-Elementen zu beeinflussen.
